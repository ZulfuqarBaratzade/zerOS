🔴 **In Development** — not ready for use yet

## Build

```bash
sudo bash scripts/build.sh
```

## Structure

- `auto/` — lb config
- `config/` — packages, hooks, files
- `scripts/` — build script
- `pkgmgr/` — future package manager
